# Инвестиционная платформа — лендинг ТКС

## О проекте

Лендинг-витрина «Технологии Коллективного Созидания» (ТКС) — направленная на развитие кооперативного движения по всей планете и продвижение модели народного софинансирования социально значимых проектов.

**Целевая аудитория:** частные инвесторы и активные партнёры, люди готовые стать пайщиками для создания и продвижения новых проектов.

**Три проекта-примера на странице:**
- **Совэлмаш** — технология совмещённых обмоток «Славянка»
- **Аэронова** — дирижабли нового поколения
- **Web3ECO** — плантации Павловнии

**Главное действие посетителя:** вступить в кооператив «ТКС» и стать партнёром или соинвестором.

## Стек

- HTML / CSS / JavaScript (статический сайт)

## Правила работы с ИИ

- Всегда общаться на русском языке
- Перед крупными изменениями спрашивать: «Хочешь сначала сделать коммит и пуш?»
- После значимых принятых действий предлагать сделать коммит и пуш
- Комментарии в коде писать на русском языке
- Объяснять термины простым языком
- Перед значительными изменениями запрашивать подтверждение
