# Анализ платформ конкурентных и референсных сайтов

Дата анализа: 30 апреля 2026 г.

---

## 1. sovelmash.ru — Совэлмаш

**Тип сайта:** Корпоративный лендинг (производство)

### Платформа
| Параметр | Значение |
|---|---|
| Конструктор | **Tilda Publishing** |
| Хостинг | Tilda CDN (облако) |
| Медиа CDN | tildacdn.com / thb.tildacdn.com |
| Видео | vkvideo.ru |
| Соцсети | VK, Telegram |

### Характеристики
- **Сложность разработки:** Низкая — собирается блоками без кода
- **Стоимость:** от 15 000 до 50 000 ₽ (фрилансер), подписка Tilda от 750 ₽/мес
- **Срок сборки:** 2–5 дней (опытный), 1–2 недели с контентом и правками
- **Масштабируемость:** Ограниченная — платформа закрытая, расширение функций упирается в возможности конструктора

### Вывод для нашего проекта
Tilda — быстро и дёшево, но не подходит для инвестиционной платформы с личными кабинетами, транзакциями и сложной логикой.

---

## 2. aeronova.pro — Aeronova (дирижабли)

**Тип сайта:** Корпоративный сайт с мультиязычностью (заказная разработка)

### Платформа
| Параметр | Значение |
|---|---|
| Фреймворк | **Nuxt.js** (на базе Vue.js) |
| CSS | Tailwind CSS v4.1.11 |
| Язык | JavaScript / TypeScript |
| Медиа CDN | Uploadcare (ucarecdn.com) |
| Дополнительно | cdn.solargroup.pro |
| Аналитика | Яндекс.Метрика |
| Шрифт | Proxima Nova |
| Мультиязычность | Да (RU / EN) |

### Характеристики
- **Сложность разработки:** Высокая — заказная разработка с нуля
- **Стоимость:** 300 000 – 1 000 000 ₽ и выше
- **Срок разработки:** от 2–3 месяцев
- **Масштабируемость:** Высокая — полноценный современный фреймворк, гибкая архитектура

### Вывод для нашего проекта
Nuxt.js — сильное техническое решение. Хорошая ссылка на уровень качества UI/UX и мультиязычности, к которому стоит стремиться.

## 3. web3eco.io — Web3Eco (эко-инвестиции)

**Тип сайта:** Маркетинговый сайт инвестиционного проекта (наиболее близкий аналог)

### Платформа
| Параметр | Значение |
|---|---|
| CMS | **WordPress 6.5.8** |
| Конструктор страниц | **Elementor + Elementor Pro** |
| Тема | Hello Elementor |
| Веб-сервер | LiteSpeed |
| Хостинг | **Hostinger** |
| SEO | Yoast SEO v22.8 |
| Аналитика | Google Analytics GA4 (MonsterInsights v9.2.4) |
| Медиа CDN | Uploadcare (ucarecdn.com) |

### Установленные плагины
- `elementor` + `pro-elements` — конструктор страниц
- `yoast-seo` — SEO
- `cookie-law-info` — GDPR cookie-баннер
- `country-code-field-for-elementor-form` — международные формы
- `social-share-buttons-lite` — кнопки шаринга
- `google-analytics-for-wordpress` — аналитика

### Характеристики
- **Сложность разработки:** Низкая–Средняя — визуальный редактор, без кастомного кода
- **Стоимость:** $5–15/мес хостинг + Elementor Pro ~$59/год; разработка 50 000–150 000 ₽
- **Срок сборки:** 1–3 недели
- **Масштабируемость:** Слабая

### Проблемы масштабируемости web3eco.io
1. Hostinger shared — упадёт при росте трафика
2. Нет кэширующего плагина — каждый запрос идёт в БД
3. Elementor генерирует тяжёлый раздутый код
4. Нет CDN для статики (JS, CSS)
5. WordPress архитектурно не подходит для Web3-функций (кошельки, смарт-контракты, токены)

### Вывод для нашего проекта
Прямой конкурент по тематике. Технически слабое решение — наш сайт уже на старте должен быть лучше по производительности, безопасности и функциональности.

## 4. reg.solargroup.pro — SOLARGROUP (инвестплатформа)

**Тип сайта:** Полноценная инвестиционная платформа с личным кабинетом — наиболее близкий функциональный аналог

### Платформа
| Параметр | Значение |
|---|---|
| Фреймворк | **Nuxt.js (SSR-режим)** на базе Vue.js |
| Аутентификация | @nuxtjs/auth, стратегия JWT (local) |
| Защита / CDN | **Cloudflare** (корпоративный уровень) |
| Медиа CDN | Uploadcare (ucarecdn.com) |
| Собственный CDN | cdn.solargroup.pro |
| SEO | Google Search Console (верификация подтверждена) |
| Размер страницы | 1.36 МБ (тяжёлое SPA) |
| Время отклика | ~3.4 сек (есть куда оптимизировать) |
| JS-бандл | 40+ чанков с code splitting (ленивая загрузка) |

### Аудитория и масштаб
- **24 языка интерфейса:** ru, en, fr, de, es, it, pl, cs, sk, hu, ro, et, lv, lt, sl, hr, bg, mk, sr, pt, zh, hi, id, vi
- Охват: Европа + СНГ + Азия + Южная Америка + Африка
- Расчётная аудитория: **сотни тысяч — миллионы** зарегистрированных пользователей
- Цели привлечения инвестиций: **100 млн $** и **от 1 млрд $**

### Функциональность (фичи)

**Инвестиционный модуль**
- Инвестиционные пакеты (покупка, увеличение, рассрочка на 3/6/n месяцев)
- Конвертация средств в инвестиционные доли
- Несколько раундов/этапов инвестирования (3-й, 5-й раунды)
- Дисконтная система и распродажи

**Финансовые операции**
- Пополнение: карты, платёжные системы, криптовалюта (бонус до +15%)
- Вывод средств (заявки, лимиты, комиссии)
- Кэшбэк-система с накоплением купонов
- Крипто-кошелёк (crypto wallet)

**Партнёрская / реферальная система (сетевая структура)**
- Реферальные ссылки и промокоды
- Многоуровневая структура (первая линия + вся структура)
- Статусы партнёров: Мастер → Эксперт → Профессионал → Президент
- Рейтинг партнёров и партнёрские гонки с призами
- Реферальные начисления (суточная статистика)
- Региональный бонус

**Бонусная система (8+ типов)**
`bonus_deposit`, `bonus_early_bird`, `bonus_double`, `bonus_conference`, `bonus_start`, `bonus_start_promo_code`, `bonus_vn_1945` и другие

**Геймификация**
- Лотереи и розыгрыши (1-й и 2-й месяц)
- Розыгрыш электромобиля и других призов
- Накопление купонов

**Личный кабинет**
- Профиль, смена пароля, редактирование данных
- Настраиваемые уведомления
- Заметки
- Документы (заказ с доставкой, апостиль)

**Контент и аналитика**
- Материалы по категориям, статьи, новости
- Аналитика и статистика по проектам
- Инструменты партнёра, результаты по этапам

**Мероприятия**
- Регистрация на вебинары, конференции, Реюньон

**Бизнес-клуб SOLARGROUP**
- Закрытый раздел для топ-партнёров

### Проекты платформы
1. **Aeronova** — дирижабли нового поколения (NOVA-2)
2. **Совэлмаш** — электродвигатели Дуюнова, производственный комплекс

### Оценка масштабируемости: **Высокая**
| Критерий | Оценка |
|---|---|
| Cloudflare DDoS-защита | Корпоративный уровень |
| Code splitting (40+ чанков) | Правильная архитектура |
| Nuxt SSR | Хорошо для SEO и первого рендера |
| Uploadcare CDN | Масштабируемое медиа |
| 24 языка | Готов к глобальному росту |
| Размер страницы 1.36 МБ | Слабое место — нужна оптимизация |
| Время отклика 3.4 сек | Медленно — есть куда расти |

### Вывод для нашего проекта
Это **главный референс** — действующая инвестиционная платформа с реальными пользователями, крипто, MLM-структурой и 24 языками. Технологический стек **Nuxt.js + Cloudflare + Uploadcare** — зрелое решение. Слабые места (тяжёлая страница, медленный отклик) — точки, где наша платформа может быть лучше с самого старта.

## Выводы для разработки нашей инвестплатформы

1. **Tilda и WordPress — не наш путь.** Не потянут функциональность инвестиционной платформы.
2. **Nuxt.js + Cloudflare — правильный стек**, подтверждённый реальной платформой (SOLARGROUP) с большой аудиторией.
3. **SOLARGROUP — главный референс по функциональности:** личный кабинет, вывод средств, крипто, партнёрская система, многоязычность.
4. **Минимальный набор с старта:** CDN (Cloudflare), HTTPS, JWT-аутентификация, аналитика, SEO-метатеги, GDPR-баннер.
5. **Критично для инвестплатформы:** безопасность, верификация пользователей, интеграция платёжных систем, крипто-кошелёк.
6. **Наши преимущества перед SOLARGROUP:** можно сделать быстрее (время отклика), легче (1.36 МБ — это много) и с более современным UX с нуля.
7. **web3eco — ориентир по контенту**, но не по технологиям. Их слабое техническое решение — наше конкурентное преимущество.
